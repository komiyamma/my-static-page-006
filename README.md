# my-static-page-006

Vite + React + TypeScript の最小プロジェクト。

## 開発

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## プレビュー

```bash
npm run preview
```

## デプロイ

- 本番公開は `npm run build` で生成される `dist/` を任意の静的ホスティングへアップロードしてください。
- 例: Azure Static Web Apps / Azure Storage 静的サイト / GitHub Pages / Cloudflare Pages など。
